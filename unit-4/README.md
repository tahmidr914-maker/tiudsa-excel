#Unit 4 – Sorting, Filtering and Tables in Excel

##Objective
This assignment demonstrates the use of Excel tables, sorting, and filtering techniques on a supermarket sales dataset.

##Tasks Completed
- Downloaded a supermarket sales dataset.
- Opened the dataset in Microsoft Excel.
- Converted the dataset into an Excel Table.
- Sorted sales data from highest to lowest.
- Applied category filters to display specific records.
- Used a custom number filter to show records above a selected sales value.
- Saved the completed workbook as an Excel (.xlsx) file.

## Skills Demonstrated
- Excel Table Creation
- Data Sorting
- Category Filtering
- Custom Number Filtering
- Data Organization and Analysis

## File Included
- sorting,filtering and tables_salesdataset.xlsx 
## Outcome
- The dataset was successfully organized and analyzed using Excel's table, sorting, and filtering features.
- improving data readability and enabling focused data analysis.
