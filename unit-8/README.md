# Unit 8 – Pivot Tables

## 📌 Overview

This project demonstrates the use of Microsoft Excel Pivot Tables to summarize and analyze a retail sales dataset. Three Pivot Tables were created to answer different business questions, and a slicer was added to provide interactive filtering.

## 📂 Dataset

- Dataset Name: Retail Sales Dataset
- Source: Kaggle
- File Format: Microsoft Excel (.xlsx)

## 🎯 Objectives

The objectives of this assignment were to:

- Create Pivot Tables for business analysis.
- Summarize total revenue by product category.
- Analyze monthly sales trends.
- Analyze customer transactions.
- Use slicers for interactive filtering.

## Pivot Tables Created

### 1. Total Revenue by Product Category

Purpose:
Displays the total sales generated by each product category.

Fields Used:
- Rows: Product Category
- Values: Sum of Total Amount



### 2. Monthly Sales Trend

Purpose:
Displays total sales for each month.

Fields Used:
- Rows: Month
- Values: Sum of Total Amount



### 3. Customer Transaction Analysis

Purpose:
Displays customer transactions based on Transaction ID count.

Fields Used:
- Rows: Customer ID
- Values: Count (or Sum, depending on browser limitations) of Transaction ID



## 🎛️ Slicer

A slicer was added to allow interactive filtering of the Pivot Tables using a selected field from the dataset.

## Skills Demonstrated

- Creating Pivot Tables
- Summarizing business data
- Sales analysis
- Monthly trend analysis
- Customer transaction analysis
- Interactive filtering using slicers
- Data organization in Excel

## Note

This assignment was completed using **Microsoft Excel for the web (browser version)**.

Due to feature limitations in Excel for the web:

- Date grouping inside Pivot Tables is not supported, so a separate **Month** column was created in the dataset.
- The **Top N (Top 5) Value Filter** option is not available in the browser version, so the Customer Transaction Analysis Pivot Table displays all customers instead of filtering only the top five.
- Some advanced PivotTable features available in the desktop version of Excel are unavailable in the browser version.

Despite these limitations, all required Pivot Tables and slicers were created using the available functionality.
