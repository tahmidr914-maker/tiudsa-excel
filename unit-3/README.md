Project: Excel / Unit 3

What I did
Created a Tax Calculator workbook in Microsoft Excel
Added Product, Price, Tax Amount, and Final Price columns
Added 15 products with prices
Used formula to calculate Tax Amount using fixed tax rate
Used formula to calculate Final Price
Created a Named Range called PriceList
Used SUM formula with Named Range for summary calculations
Calculated Total Price, Total Tax, and Total Final Price

Files / links

Tax_Calculator.xlsx

How to run / verify

Open Tax_Calculator.xlsx

Check columns C and D for calculated Tax Amount and Final Price
Check cell F2 for tax rate value
Check Name Box for named range "PriceList"
Check summary section in cells F5:G7

Notes / references

Tax rate used: 10%
Absolute reference ($F$2) used for fixed tax rate
Named Range "PriceList" used in formulas
All calculations are formula-driven

Excel formulas used:
= B2*$F$2
= B2+C2
= SUM(PriceList)
= SUM(PriceList)*$F$2
= G5+G6
