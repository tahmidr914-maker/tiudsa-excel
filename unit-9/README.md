# Unit 9 - Charts and Data Visualization

## Overview

This project demonstrates the use of Excel charts and data visualization techniques to analyze retail sales data generated in Unit 8.

## Charts Created

### 1. Revenue by Product Category

A bar chart was created to compare revenue generated by each product category.

### 2. Monthly Sales Trend

A line chart was created to visualize sales performance over time and identify trends across different months.

### 3. Revenue Share Distribution

A pie chart was created to display the contribution of each product category to total revenue.

### 4. Revenue vs Order Count

A combination chart using columns and a line graph was created to compare revenue and order count simultaneously using dual axes.

## Formatting Improvements

* Added chart titles and axis labels.
* Applied custom chart formatting and color selections.
* Configured a secondary axis for the combo chart.
